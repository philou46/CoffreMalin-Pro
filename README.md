# 🔐 CoffreMalin Pro

**CoffreMalin Pro** est un gestionnaire de mots de passe local, simple et sécurisé, conçu pour stocker ses identifiants dans un coffre chiffré, sans cloud imposé.

L’objectif est clair : garder ses mots de passe organisés, protégés et accessibles uniquement par l’utilisateur.

---

## ✨ Fonctionnalités principales

- 🔐 Coffre local chiffré
- 🧠 Mot de passe maître unique
- 🔑 Générateur de mots de passe sécurisés
- 📊 Audit de sécurité des identifiants
- 🗂️ Organisation par catégories
- 💾 Sauvegardes du coffre
- 📥 Import depuis Bitwarden
- 📤 Export possible des données
- 🆘 Fiche d’urgence pour ne pas perdre l’accès à ses informations importantes
- 🖥️ Application Windows simple à utiliser

---

## 🛡️ Sécurité

CoffreMalin Pro est pensé pour une utilisation locale.

Les données sensibles sont stockées dans un coffre chiffré.  
L’utilisateur reste responsable de son mot de passe maître : il ne doit pas être oublié, car il permet d’accéder au coffre.

> ⚠️ Aucun logiciel ne peut protéger efficacement des mots de passe si le mot de passe maître est faible, partagé ou conservé n’importe où.

---

## 📦 Installation

1. Télécharger le fichier d’installation depuis la page **Releases**
2. Lancer l’installateur
3. Ouvrir CoffreMalin Pro
4. Créer un mot de passe maître
5. Commencer à ajouter ses identifiants

👉 Les versions téléchargeables se trouvent dans l’onglet **Releases** du dépôt GitHub.

---

## 🧩 Import Bitwarden

CoffreMalin Pro permet d’importer une sauvegarde Bitwarden afin de faciliter la migration depuis un autre gestionnaire de mots de passe.

Avant tout import, il est recommandé de faire une sauvegarde de son coffre actuel.

---

## 💾 Sauvegardes

Il est fortement conseillé de réaliser régulièrement une sauvegarde du coffre.

Une sauvegarde permet de restaurer ses données en cas de problème, de changement d’ordinateur ou de suppression accidentelle.

---

## ⚠️ À propos des exports

Certains formats d’export, comme le CSV, peuvent contenir des informations sensibles en clair.

Il est conseillé de supprimer immédiatement tout fichier d’export non chiffré après utilisation.
